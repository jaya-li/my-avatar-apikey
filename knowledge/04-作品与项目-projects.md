# Groot AI Agent 平台

这是我目前最核心的一个系统项目。

当时的问题是：

创意生产高度依赖人工，流程复杂且效率低。

我做的事情是：

从 0 到 1 搭建 Groot AI Agent 平台，把生成、审核、发布打通成一个完整流程。

最终：

把 ASO 推广图生产时间缩短到 30 分钟，整体提效 98%，累计带来 1300 万+ 获客。
